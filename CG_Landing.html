    • The purpose of this project was to analyze weather changes as you get closer to the equator. To accomplish this anlaysis, we first pulled random latitude and longitude data points. Next, we used the OpenWeatherMap API to assemble a weather dataset for 500 cities.


    Matplotlib was used to plot various aspects of weather and its relationship to latitude. Weather comparisons include: temperature, humidity, cloudiness and wind.

    Select Plots from the dropdown menu in the toolbar to see the results!
